# 돈가스 지도 관리자 페이지

React + Vite로 구현한 돈가스 지도 백오피스 어드민 페이지입니다.

## 시작하기

### 1. 패키지 설치
```bash
npm install
```

### 2. 환경변수 설정
`.env` 파일을 생성하고 관리자 계정 정보를 설정하세요:

```env
VITE_ADMIN_USERNAME=your_username
VITE_ADMIN_PASSWORD=your_password
```

### 3. 개발 서버 실행
```bash
npm run dev
```

### 4. 빌드
```bash
npm run build
```

## 기능

- 반응형 로그인 페이지 (모바일/데스크톱 지원)
- 환경변수 기반 인증

## 보안 관련 주의사항

### 현재 구현의 보안 수준

현재 구현은 **기본적인 접근 제어**만 제공하며, **실제 프로덕션 환경에서는 안전하지 않습니다**:

#### 보안 문제점:
1. **클라이언트 사이드 인증**: 환경변수(`VITE_*`)는 빌드 시 번들에 포함되어 브라우저에서 접근 가능
2. **자격증명 노출**: 브라우저 개발자 도구에서 소스 코드를 보면 아이디/비밀번호 확인 가능
3. **세션 관리 없음**: 페이지 새로고침 시 로그인 상태 유지 안됨
4. **암호화 없음**: 비밀번호가 평문으로 비교됨

#### 개선 방안:
1. **백엔드 API 구축**: Node.js/Express, Django 등으로 서버 구현
2. **JWT 토큰 사용**: 로그인 성공 시 토큰 발급 및 관리
3. **HTTPS 사용**: 통신 암호화
4. **비밀번호 해싱**: bcrypt 등으로 비밀번호 암호화
5. **세션/쿠키 관리**: 로그인 상태 유지
6. **환경변수 분리**: 서버 환경변수 사용 (클라이언트 노출 방지)

### 권장 사용 방법

- **내부 네트워크 전용**: VPN이나 사내망에서만 접근 가능하도록 설정
- **임시 테스트용**: 프로토타입이나 로컬 개발 환경에서만 사용
- **프로덕션 배포 시**: 반드시 백엔드 인증 시스템 구축 필요
